# summernote_demo

> summernote 中整合百度公式编辑器

## 使用

由于kityformula部分文件读取问题，需放在服务器上进入index.html使用

## 目录说明
> * `kityformula-plugin` 为百度公式编辑器
>
> * `plugin/kityplugin/summernote-ext-kity.js` 为增加的公式编辑器按钮逻辑
>
